# SecurePass Pro

O SecurePass Pro é um gerador de senhas com interface gráfica desenvolvido em Python utilizando CustomTkinter.

#Funcionalidades

- Definição do tamanho da senha
- Opção de incluir letras maiúsculas, números e símbolos
- Indicador visual de força da senha
- Botão para copiar senha para a área de transferência
- Armazenamento local das últimas 10 senhas geradas (JSON)

#Tecnologias utilizadas

- Python
- CustomTkinter
- Manipulação de arquivos (JSON)
- Biblioteca random

#Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, desenvolvimento de interface gráfica e conceitos básicos de segurança da informação.

#Observação sobre segurança

As senhas são armazenadas localmente apenas para fins de demonstração.
